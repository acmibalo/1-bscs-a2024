Game Details

So I made this game to not fail my class. Game is widely inspired by the scary maze game meme that was popular back in early 2020's. It was known for scaring you when nearing the end of the game. I tried to do the same with this game but changed stuff since I'm a beginner programmer and have no clue what I'm doing most of the time.

Instead of having multiple levels, I chose to have only one decently sized level. And instead of a jumpscare only when nearing the end, I chose to have a rickroll when reaching the end and touching any of the walls would proc the jumpscare instead.

Still a work in progress.... I think.
